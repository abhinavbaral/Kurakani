# Kurakani

**Kurakani**, ("conversation" in Nepali), is a web-based real-time chat app that allows users to send messages , join a shared group chat, and many more features all in real time.

### Tech Stack

- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: Postgress
- **Deployment**:Vercel
